# كابينة مونديال

تطبيق عربي RTL لإدارة محتوى فريق إنتاج فيديوهات كأس العالم بدون قاعدة بيانات. البيانات تحفظ محليًا في المتصفح باستخدام `localStorage`، مع تصدير واستيراد JSON للنسخ الاحتياطي والنقل بين الأجهزة.

## التشغيل

افتح `index.html` مباشرة في المتصفح.

لتحميل بيانات تجريبية في أول تشغيل فقط:

```text
index.html?demo=1
```

## سير العمل اليومي

1. افتح صفحة **المحرر** وأضف أخبار اليوم أو ملخصات المباريات.
2. استخدم **عرض الفريق** لقراءة المحتوى ونسخ الملخص أو السكريبت السريع.
3. من أي خبر، اضغط **إنشاء بوستر الجريدة** لتوليد تصميم عربي كلاسيكي.
4. احفظ البوستر ليظهر في **معرض البوسترات**.
5. في نهاية اليوم، استخدم **تصدير JSON** لحفظ نسخة احتياطية.

## النسخ الاحتياطي

- زر **تصدير JSON** ينزّل نسخة كاملة من الأخبار والبوسترات.
- عند اختيار **استبدال الحالي** أثناء الاستيراد، يقوم التطبيق بتنزيل نسخة احتياطية تلقائيًا قبل الاستبدال.
- الصور تحفظ داخل `localStorage` بعد ضغطها، لكن الأفضل حذف البوسترات والصور القديمة عند ارتفاع مساحة التخزين.

## ملاحظات إنتاجية

- التطبيق لا يحتاج backend أو قاعدة بيانات.
- تصدير PNG يعمل من ملف `vendor/html2canvas.min.js` المحلي، لذلك لا يعتمد على CDN خارجي.
- لو هتربط مساعد الصياغة بذكاء اصطناعي حقيقي لاحقًا، استخدم backend بسيط لحماية مفتاح API.

## التطبيق كـ PWA (تثبيت + أوفلاين)

- التطبيق قابل للتثبيت على سطح المكتب/الهاتف (manifest + service worker)، ويعمل بالكامل بدون إنترنت بعد أول فتح.
- كل الخطوط العربية وأداة تصدير الصور (`html2canvas`) محفوظة محليًا داخل المشروع — لا يعتمد على أي CDN.
- ملاحظة: الـ service worker يعمل فقط عبر `http(s)`؛ عند فتح `index.html` مباشرة بـ`file://` يعمل التطبيق لكن دون تثبيت/كاش.

## النشر على الإنترنت (GitHub Pages)

التطبيق ملفات ثابتة بالكامل (لا build)، فيمكن نشره مباشرة:

1. ارفع المجلد إلى مستودع GitHub.
2. من إعدادات المستودع: **Settings → Pages** اختر **Deploy from a branch**.
3. اختر فرع `main` والمجلد `/ (root)`، أو فرع `gh-pages` والمجلد `/ (root)`.
4. بعد أول نشر، الموقع سيكون متاحًا هنا:

```text
https://beshoadel7878-prog.github.io/kabint-mondial/
```

ملف `.nojekyll` موجود ليضمن خدمة الملفات كما هي. كل المسارات نسبية فيعمل التطبيق من أي مسار فرعي.

## اختصارات لوحة المفاتيح

- `Alt`+`1..6`: التنقل بين الأقسام
- `N`: خبر جديد · `/`: تركيز البحث · `Ctrl`+`S`: حفظ (المحرر/البوستر) · `?`: المساعدة · `Esc`: إغلاق النوافذ
